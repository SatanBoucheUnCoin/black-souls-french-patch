# BLACK SOULS - Patch de Traduction Française

Traduction française non-officielle pour **BLACK SOULS** (RPG Maker VX Ace).

## Installation

1. Localisez votre dossier d'installation de BLACK SOULS :
   - **Steam (Linux)** : `~/.local/share/Steam/steamapps/common/BLACK SOULS/`
   - **Steam (Windows)** : `C:\Program Files (x86)\Steam\steamapps\common\BLACK SOULS\`

2. Copiez le dossier `Data/` de ce patch dans le dossier du jeu.
   - Si un dossier `Data/` existe déjà, fusionnez les fichiers (écrasez les existants).
   - Sinon, copiez simplement le dossier `Data/` tel quel.

3. Lancez le jeu !

### Désinstallation

Supprimez le dossier `Data/` du jeu. Le jeu utilisera automatiquement les fichiers originaux contenus dans `Game.rgss3a`.

## Problèmes connus

Si le jeu ne charge pas la traduction, vérifiez que :
1. Le dossier `Data/` est bien placé à la racine du jeu (à côté de `Game.exe`)
2. Les fichiers `.rvdata2` sont directement dans `Data/`, pas dans un sous-dossier
